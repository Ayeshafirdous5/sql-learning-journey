# Day 2 - WHERE Clause

## Topics Covered

- WHERE Clause
- Comparison Operators
- Logical Operators
- LIKE Operator
- Wildcards (% and _)

## Comparison Operators

- =
- !=
- >
- <
- >=
- <=

## Logical Operators

- AND
- OR
- NOT

## LIKE Wildcards

- % → Any number of characters
- _ → Exactly one character

## What I Learned

The WHERE clause is used to filter rows based on a condition.

I also learned how to combine multiple conditions using AND, OR, and NOT.

The LIKE operator helps search for patterns using % and _ wildcards.
