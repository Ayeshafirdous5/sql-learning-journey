## 🎯 Key Learning

- **WHERE** filters individual rows before grouping.
- **HAVING** filters grouped results after `GROUP BY`.
- **LIMIT** restricts the number of rows returned.
- **LIMIT offset, count** is useful for pagination.
- **AS (Alias)** makes query results more readable.
- Combining multiple SQL clauses helps solve real-world data analysis problems.
