# Day 1 - SELECT & DISTINCT

## What I learned

- SELECT is used to retrieve data from a table.
- * returns all columns.
- DISTINCT returns only unique values.

## Queries Practiced

- SELECT *
- SELECT first_name
- SELECT last_name
- SELECT first_name, last_name
- SELECT salary
- SELECT department
- SELECT DISTINCT department
- SELECT DISTINCT city
- SELECT DISTINCT gender
