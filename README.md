# 🚀 SQL Learning Journey

Welcome to my SQL Learning Journey!

This repository documents my step-by-step journey of learning SQL for Data Analytics through hands-on practice. Instead of only watching tutorials, I solve queries, practice them on my own dataset, and document everything here.

My goal is to build a strong SQL foundation while creating a portfolio that showcases consistent learning and practical skills.

## 📂 Database

I am using a custom **Online Store** database created specifically for practicing SQL concepts.

The database includes employee information such as:

- Employee ID
- First Name
- Last Name
- Gender
- Department
- City
- Salary
- Joining Date
